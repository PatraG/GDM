# Appwrite Setup Instructions

## Database Setup

### 1. Create Appwrite Database

**Task**: T006 - Create Appwrite database `oral_health_survey`

**Steps**:
1. Log in to Appwrite Console: https://cloud.appwrite.io
2. Navigate to your project (Project ID: 691351eb00227fd3a6ea)
3. Go to "Databases" in the left sidebar
4. Click "Create database"
5. Enter database name: `oral_health_survey`
6. Database ID: Use auto-generated or set to `oral_health_survey`
7. Click "Create"

**Verification**:
- Database appears in the databases list
- You can navigate into the database
- Database ID is recorded for environment variables

**Next Steps**: Continue with collection creation (T017-T024)
